# ROLEX

## 1. 구현 목표

-   관리자, 회원, 게시판이있는 쇼핑몰 구현 


## 2. 개발 일정

-   2021.10.31 ~ 2021.12.07

## 3. 개발환경

개발언어 JAVA / JSP / HTML5 / CSS / JavaScript / Ajax / Jquery

데이터베이스 My sql 5.1.49 / HeidiSql 9.5
